Sobre Mim


Front end

É a parte visual de um site, aquilo que conseguimos interagir. Quem trabalha com Front End é responsável por desenvolver por meio do código uma interface gráfica e, normalmente, com as tecnologias base da Web: HTML, CSS e JavaScript.


Back end

Back-End trabalha em boa parte dos casos fazendo a ponte entre os dados que vem do navegador rumo ao banco de dados e vice-versa, sempre aplicando as devidas regras de negócio, validações e garantias num ambiente restrito ao usuário final (ou seja, onde ele não consegue acessar ou manipular algo).


Full stack

Full Stack developer é quem trabalha com Front End e Back End.


Analista de sistemas

O analista de sistemas desempenha um papel crucial na indústria de Tecnologia da Informação, utilizando sua expertise para criar programas personalizados que atendam às necessidades exclusivas das empresas. Basicamente, ele é encarregado de projetar e desenvolver soluções inovadoras que resolvam os desafios enfrentados pelas organizações modernas, por meio de sistemas computacionais avançados.
